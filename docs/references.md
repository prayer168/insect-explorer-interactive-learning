## 概念正確性說明
- 昆蟲特徵：身體三部分（頭、胸、腹）、3 對（6 隻）腳，多數有翅，屬節肢動物門昆蟲綱。
- 蜘蛛（蛛形綱，8 腳、身體兩部分）、蜈蚣（多足類）、蝸牛（軟體動物）皆非昆蟲。
- 完全變態：卵 → 幼蟲 → 蛹 → 成蟲（蝴蝶、甲蟲、蜂、蠅、瓢蟲等）。
- 不完全變態：卵 → 若蟲 → 成蟲（蝗蟲、蟋蟀、蟑螂、蜻蜓等）。
- 課綱代碼為對應建議，實際代碼以正式課綱手冊為準。
