# 設計規格：昆蟲大解密

## 設計原則
- 教學正確優先，動畫採「教學寫實」而非炫技。
- 每頁一個主要學習任務，資訊分區、留白充足。
- 內容（JSON）與程式邏輯分離，便於維護與更新題庫。

## 視覺
- 主色：自然綠（#2f7d4f / #4caf72）、葉綠、天藍、太陽黃點綴。
- 字體：Noto Sans TC，內文約 18px，行高 1.8。
- 圓角卡片 + 柔和陰影，按鈕大且適合觸控（最小高度約 44px）。
- 對比符合可讀性，狀態不只用顏色，另加文字與符號（✅❌）。

## 資訊架構（7 頁籤）
學習任務 → 昆蟲在哪裡 → 昆蟲如何長大 → 昆蟲重要嗎 → 生活應用 → 闖關大挑戰 → 自主學習

## 互動元件
| 元件 | 技術 | 操作 | 無障礙 |
|---|---|---|---|
| 找昆蟲遊戲 | DOM + CSS 動畫 | 點擊判斷 | 按鈕可聚焦、aria-label、有重設 |
| 數腳分類 | 拖放 + 點選 | 拖曳或先選再點桶 | 鍵盤 Enter/Space、觸控相容 |
| 變態動畫 | 行內 SVG + JS | 播放／暫停／單步／重設 | 文字標示階段、可單步 |
| 授粉模擬 | 行內 SVG + CSS | 點花朵授粉 | tabindex、鍵盤可操作 |
| 闖關測驗 | JSON 驅動 | 作答即回饋 | 按鈕語意、解析文字 |

## 資料檔
- `data/content.json`：各單元教學文字。
- `data/quiz.json`：闖關題庫（題型、答案、解析、回看單元）。
- `data/resources.json`：自主學習資源。
- `data/curriculum.json`：課綱對應。

## 技術
- 純前端，無框架；以相對路徑（`./`）確保 GitHub Pages 子路徑可運作。
- localStorage 記錄學習進度。
- 響應式斷點：768px（平板）、520px（手機）、360px（小手機）。
